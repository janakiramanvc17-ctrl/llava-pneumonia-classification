# llava-pneumonia-classification
# LLaVA Pneumonia Classification (Vision-Language Model)

This project evaluates LLaVA-1.5-7B as a zero-shot classifier for pneumonia detection
on PneumoniaMNIST.

## Model

- LLaVA-1.5-7B (Vision-Language Model)
- Loaded from Hugging Face
- Zero-shot inference (no fine-tuning)

## Dataset

- PneumoniaMNIST
- 624 test samples

## How to Run

```bash
python evaluate_llava.py
